📞 Contact Book (Python + JSON)

A simple command-line Contact Book application built using Python. This project allows users to store, search, edit, and delete contacts. All contact information is saved in a JSON file, so the data persists even after the program is closed.

✨ Features :
~ Add new contacts
~ View all saved contacts
~ Search contacts by name
~ Search contacts by phone number
~ Edit existing contacts
~ Delete contacts

📖 MENU
1. Add Contact
2. Show All Contacts
3. Delete Contact
4. Search Contact
5. Edit Contact

PROJECT STRUCTURE :
PHONE_BOOK 
│
├── contacts.json      # Stores all contact data
├── main.py            # Main Python program
└── README.md          # Project documentation
